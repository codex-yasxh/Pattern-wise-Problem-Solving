# DSA from Scratch

This repository contains my Data Structures & Algorithms solutions written as complete C++ programs.

Instead of relying on online judge templates, every problem is implemented from scratch, including:

* Creating custom data structures (when needed)
* Building test cases
* Calling functions through `main()`
* Printing and verifying outputs

The goal is to practice writing interview-ready code that can run independently in any C++ environment, not just on platforms like LeetCode.

## Structure

* Two pointers
* Hashing
* Sliding Window
* Prefix Sum & Difference Arrays
* Matrix/2D Grid Traversals
* Kadane & Subarray Extremes
* Monotonic Stack & Queues
* Linked Lists
* Sorting + Greedy
* Intervals
* Trees
* Graphs
* Dynamic Programming
* Greedy
* Backtracking
* and more...

## Purpose

* Strengthen DSA fundamentals
* Prepare for coding interviews
* Write production-like C++ instead of platform-dependent solutions
* Build a personal reference of solved problems
